# 실행

yarn add
yarn start

# URL

1. /analysis : 분석 페이지
2. /friendsInvite : 친구 초대
3. /friendsMessage : 친구에게 메세지 보내기
4. /friendsMessage/Gift : 메세지에 아이템 선택
5. /myItem : 내 아이템
6. /myItem/pick : 내 아이템 뽑기

# 기술 스택

1. TypeScript (4.9.5)
2. React.js (18.2.0)
3. React-route-dom (6.20.1)
4. zustand (4.5.2)
